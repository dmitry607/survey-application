# Survey Application - Wpf

## Описание
приложение для проведения опросов с подсчетом баллов и отправкой результатов на электронную почту

## требования
- .NET 8.0+
- VS 2022

## Настройка
1. откройте settings.json
2. измените настройки SMTP

## Запуск
Откройте SurveyApp.sln в Visual studio, затем нажмите F5

## Технологии
- C# 12
- WPF
- MVVM
- MailKit
- конфигурация System.Text.Json
